== Description ==
This plugin sends alerts to designated email addresses when new posts are pending admin approval. 

== Installation ==
1. Copy the entire /pendingpostalert/ directory into your /plugins/ directory.
2. Activate the plugin.
3. Go to plugin settings page in the admin area and create a comma separated list of email addresses to by notified when new posts are pending admin approval.
